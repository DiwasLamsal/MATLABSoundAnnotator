Sound annotator using MATLAB

This application was made using MATLAB app designer
The main application can be executed by running the startup.m MATLAB script file 

Editing the code in app designer requires newer version of MATLAB. 
The SoundAnnotatorMain.mlapp is the main application which requires MATLAB version 2018 or newer. 
SoundAnnotatorMain2016.mlapp can be used to edit the application in MATLAB version 2016.
